# Deeplus

Deeplus is an early-stage programming language project.

## Project status

The project is currently being initialized. Its language design, implementation
strategy, toolchain, and release process have not been finalized yet.

There is no compiler, interpreter, package manager, or stable language
specification available at this stage.

## Initial roadmap

The project will develop incrementally through tracked issues and focused pull
requests. The first milestones are expected to cover:

1. Defining the language goals and design principles.
2. Drafting a minimal language specification.
3. Selecting an implementation strategy and toolchain.
4. Building a small executable prototype.
5. Adding automated tests and usage examples alongside the implementation.

Roadmap items describe the intended sequence of work, not finalized design
decisions. Concrete proposals should be discussed in GitHub issues before they
are implemented.

## Contributing

Deeplus is not yet ready for general-purpose use, but focused design and
implementation contributions are welcome. Read [CONTRIBUTING.md](CONTRIBUTING.md)
before opening a pull request.
